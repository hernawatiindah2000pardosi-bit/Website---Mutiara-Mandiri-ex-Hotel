# Mutiara Mandiri ex-Hotel Website

Hasil proyek tim Web Design yang membangun sebuah website responsif berkonsep modern untuk bisnis thrift furnitur eks-hotel.

## 🌟 Fitur & Keunggulan
- **Responsive Layout:** Tampilan adaptif yang dioptimalkan sepenuhnya untuk kenyamanan akses melalui smartphone, tablet, maupun desktop.
- **UI/UX Modern:** Desain antarmuka bersih yang disesuaikan untuk meningkatkan minat konversi pelanggan ritel furnitur.

## 🛠️ Teknologi yang Digunakan
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)[cite: 1]

## 👥 Kontribusi Tim
- Mengembangkan struktur semantik halaman web[cite: 1].
- Memastikan responsivitas tata letak di setiap ukuran resolusi layar browser[cite: 1].
